# Pico Neo 2 WiVRn
A native VR client for the Pico Neo 2 and Pico Neo 2 Eye that allows PCVR gameplay over WiFi and USB for Linux.

## Supported headsets
- Pico Neo 2
- Pico Neo 2 Eye* (not tested no eye tracking)
